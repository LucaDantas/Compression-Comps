# Details for pipeline documentation

## Input
Receives a filename as an argument from the command line. Make sure it is in some lossless format such as .png or .tiff. Read in the image into a variable in RGB space and get ready for pipeline.

## Conversion to YCbCr
Map the data from RGB space to YCbCr space following the JPEG standard

Optional: downsampling the YCbCr coefficients.
